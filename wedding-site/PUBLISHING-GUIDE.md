## Option 1: GitHub Pages (FREE & Easy) ‚≠ê RECOMMENDED

**Best for:** Free hosting, custom domain support, automatic HTTPS

### Steps:
1. **Create a GitHub account** (if you don't have one): https://github.com/signup

2. **Create a new repository:**
   - Go to https://github.com/new
   - Name it: `wedding-website` (or anything you like)
   - Make it **Public**
   - Click "Create repository"

3. **Upload your files:**
   - Click "uploading an existing file"
   - Drag and drop your `index.html` file
   - Click "Commit changes"

4. **Enable GitHub Pages:**
   - Go to repository Settings ‚Üí Pages (left sidebar)
   - Under "Source", select "Deploy from a branch"
   - Choose branch: `main`, folder: `/ (root)`
   - Click "Save"

5. **Your site will be live at:**
   - `https://YOUR-USERNAME.github.io/wedding-website/`
   - It takes 1-2 minutes to deploy

### To add a custom domain (like jordanandmaggie.com):
1. Buy a domain from Namecheap, Google Domains, or GoDaddy
2. In your domain settings, add a CNAME record pointing to `YOUR-USERNAME.github.io`
3. In GitHub Pages settings, add your custom domain
4. Enable "Enforce HTTPS"

**Best for:** Drag-and-drop simplicity, automatic deployments

### Steps:
1. **Go to:** https://app.netlify.com/drop

2. **Drag your entire `wedding-site` folder** onto the page

3. **Your site goes live instantly!**
   - You'll get a URL like: `https://random-name-12345.netlify.app`
   - You can change the name in Site Settings

### To add a custom domain:
1. Click "Domain settings" ‚Üí "Add custom domain"
2. Follow the DNS setup instructions

## üì± Testing on Mobile

Before publishing, test on your phone:
1. Open `index.html` locally on your computer
2. Find your computer's local IP address:
   - Mac: System Preferences ‚Üí Network

Or just publish it and test the live URL!
